In ASSETS, go to:  Home > Tasks > Worker Alerts

The Worker Alerts screen is like a room.

In this room, we have five file cabinets. In the ASSETS system, we refer to these file cabinets as "Modes."

The five modes are:

    • Enforcement (mode)

    • IVA (mode)

    • Management (mode)

    • Collections (mode)

    • COLA (mode)

Within each mode, we have a number of tabs. The tabs displayed change according to which mode is selected. The tabs are like file folders inside a file cabinet.

Reason Codes

MEDX

The MEDX tabs (there are three of them) track medical executions sent to employers. It (what, exactly?) functions like delinquency switches, but in regard to medical support.

Most non-informational messages under the MEDX-related tabs signify that some kind of medical-related service is being held up for some reason.

MEDX With Exceptions

 

MEDX Variable Data

This tab is informational only. It contains no actionable messages or mechanisms for taking action directly from that screen.

PLS

The PLS tab serves as an umbrella for hits returned via various location-related or enforcement-related databases.

These databases/resources include:

    • DOD

    • DMV

    • FBI

    • IRS

    • NDNH

    • SSA

    • Taxation and Finance

    • VA (Department of Veterans' Affairs)

With the exception of information provided by the Social Security Administration, new information regarding a case that is displayed on the PLS tab (e.g. new address, new hire reporting, etc etc) will have been automatically entered into the case record.

When workers performing location-related activities run queries through various location-related databases, any hits returned or new information (e.g. new address, new hire reporting, etc etc) hits returned regarding those searches will appear on the PLS tab, Enforcement mode.

FCR

Contains mostly information about social security numbers being confirmed or rejected.
