Continuation of Services when TA benefits cease

Each week, the WMS system notifies the ASSETS system of temporary assistance cases that have been closed by the TA Unit.

This happens via transmission of a "IVA-IVD File" file from WMS to ASSETS.

Specifically, each week:

    1. ASSETS receives the IVA-IVD file from WMS.

    2. The IVA-IVD file notifies ASSETS of temporary assistance case that have been closed by the TA Unit during the past week.

    3. ASSETS generates data files to be used in constructing Continuation of Services letters.

    4. ASSETS sends the data files to the Processing Center.

    5. The Processing Center prints and mails the Continuation of Services letter to CPs and NCPs.
