Screenshots — Requirements and Parameters

Capture the Screenshots

    • Capture images with SnagIt

    • Resolution set at 300 dpi

    • No border

    • No shadow

    • Images must be IMPORTED into RoboHelp — not just copied and pasted in.  If you copy and paste an image into a RoboHelp page, RoboHelp will auto-assign a name to it like "image1.gif." That's no good. Images must have meaningful file names.

Format the Images WITHIN RoboHelp

In RoboHelp, format the images that you have imported into a topic page. To do that, right-click on the image and choose "Image Properties."  The dialog box pictured below will appear.

    • Include Screen tip and ALT text for every image.

    • Add borders: Black, 1 px
