Glossary Issues

There are the following issues with the glossary:

Problem

The glossary hotspot wizard is supposed to be case-sensitive, but that mechanism is malfunctioning.

  1. The word "can" (e.g. I can come tomorrow) is being picked up as CAN.

  2. The word "assets" is being picked up by ASSETS.

Solution

There are two solutions to this:

Do not run the hotspot wizard globally, in "All Folders" and "Any Status."

Instead, do it folder by folder, or use the status setting.
