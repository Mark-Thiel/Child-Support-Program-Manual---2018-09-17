Tips for Using the Program Manual

 

  1. Quickly find the meaning of any individual Status Code

    Type any individual Status Code into the Search box of the online Child Support Program Manua, then hit the Enter key.   

    (Example: P208 + Enter, or L778 + Enter)

  2. Look up the meaning of each group of Status Codes

    Find the meaning of each group of Status Codes in the Appendices > Code Tables > Group Codes and their Meanings section of this manual.
