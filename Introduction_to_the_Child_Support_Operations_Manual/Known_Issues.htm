Known Issues with this Manual

Previous and Next buttons are missing when using Internet Explorer

Problem:  The "Previous" and "Next" buttons (green arrows) do not appear when viewing the manual via Microsoft Internet Explorer.  They appear perfectly when using Google Chrome or Mozilla Firefox browsers.

Solution:  Use Google Chrome or Mozilla Firefox.

OR

Use keyboard shortcuts to advance to the next page. Use the tab key to highlight the next page in the Table of Contents, then hit the Enter key to go to that page.

 

Active pages are not highlighted in TOC when using Internet Explorer

Problem:  The active page is not highlighted in the Table of Contents when viewing the manual via the Microsoft Internet Explorer browser.

Solution:  Use Google Chrome or Mozilla Firefox browsers.
