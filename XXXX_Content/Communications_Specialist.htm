My latest work product is the New York State Child Support Program Manual. ) contains a wealth of information, as well as major conveniences for child support workers and administrators, including:

  • Videos

  • Desk Aids

  • Quick Reference Guides

  • Clickable, interactive process diagrams

  • Downloadable PDFs of each individual chapter

  • Illustrated, step-by-step instructions for procedures

  • Pop-up code tables wherever codes are mentioned

  • Direct links to documents wherever documents are mentioned

  • Pop-Up screenshots wherever ASSETS screens are mentioned

Mark Thiel designed, built, researched, wrote, illustrated, edited, and published the manual to be a fully interactive online help system for both recently-hired and seasoned workers alike
