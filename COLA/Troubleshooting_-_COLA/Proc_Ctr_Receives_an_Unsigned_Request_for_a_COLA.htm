Processing Center Receives an Unsigned Request for a COLA

For a request for a COLA to be entered into the system by the Processing Center, the COLA Request Form must be signed by the requesting party.

If the COLA Request Form isn't signed, no request is entered into the system.

Instead, the Processing Center scans the unsigned COLA Request Form and sends a copy via ECS to the local SCU.

The SCU will reach out to the party in question in order to obtain the necessary signature.
