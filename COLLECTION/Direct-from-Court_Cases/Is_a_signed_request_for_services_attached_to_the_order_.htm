Is a signed request for services attached to the order?

A signed request for services may be in the form of:

  • petition (signed prior to hearing)

  • addendum (signed at the hearing) — Form 4-3b, Addendum to Support Petition — Request for Child Support (IV-D) Services. (See 03 ADM 5 for details)

  • motion (supreme court)

If no signed request is attached to the order, the worker must check the Court Interface, and look for a petition, addendum, or motion.

If one is found, the worker starts building the case.

If not found, the worker mails a 4882 to the CP. (Do not start building case until you have a valid, signed request for services.
