Wage Order Delinquency — Set a Value

The Wage Order Delinquency field on the XXXX > XXXX screen cannot be blank or contain a zero.

The worker must enter a value of one dollar ($1.00) or greater -- otherwise an IWO will not be sent out for this case.

The reason for this is that prior to [INSERT DATE and/or legislation here], the issuance of an IWO was purely an enforcement (as opposed to collection) action, and was taken when NCP's failed to make agreed-upon payments.

In other words, in the past,IWOs were issued only on deliquent accounts -- and therefore, the system (ASSETS/CSMS) technologically required that there be a value of $1.00 or greater in the Wage Order Delinquency field.

Today, the policy (of issuing IWOs only to delinquent accounts)  is gone — but the technological requirement remains.  

These days, issuing anIWO, and having the child support payments deducted from paychecks is standard operating procedure for ALL cases, when possible.

    • The Child Support Program sends an IWO to an employer

    • The employer deducts the required child support payment from the NCP's paycheck

    • The employer sends the child support payment to the Child Support Program

For more information, see the section of this manual on Income Withholding Orders.
