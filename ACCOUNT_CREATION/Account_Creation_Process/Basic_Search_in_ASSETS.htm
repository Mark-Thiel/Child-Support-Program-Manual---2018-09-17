ASSETS Basic Search

  1. Go to: Modules > Inquiry

     

The Basic Search screen appears.

 

 

 

  1. Search by any of the following criteria:

 

 

  • CP's ID#

 

 

  • NCP's ID#

 

  • CP's Social Security number

 

  • NCP's Social Security number

 

 

  • Case ID#

 

  • Name of Child

 

The system will return hits for the name entered, but also for any names that are similiar. For example, if you search for "Joe Smith," ASSETS will return hits for:

  • Joe Smith

  • Jo Smith

  • Joseph Smith

  • Josef Smith

  • Jose Smith

  • Name of Customer

 

  • Name of PF/NCP
