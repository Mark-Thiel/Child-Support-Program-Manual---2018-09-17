To access the ACS on the Family Court Interface:

    1. Select Modules.

    2. Select Court from the drop-down menu.

    3. Select Messages from the drop-down menu.

The Court screen, Messages tab appears.

 

    1. Enter Docket Number or Case ID if available; otherwise select the ACS from the Document field and the appropriate District in the District field.

    2. Click the Search button.
