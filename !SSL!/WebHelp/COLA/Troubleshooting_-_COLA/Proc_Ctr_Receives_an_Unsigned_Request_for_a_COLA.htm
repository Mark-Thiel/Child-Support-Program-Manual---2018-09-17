Processing Center Receives an Unsigned Request for a COLA Cost-of-Living Adjustment, sometimes called a Cost-of-Living Allowance.

For a request for a COLA to be entered into the system by the Processing Center, the COLA Request Form must be signed by the requesting party.

If the COLA Request Form isn't signed, no request is entered into the system.

Instead, the Processing Center scans the unsigned COLA Request Form and sends a copy via ECS Electronic Communications System, used for communications between the Processing Center and local SCUs. to the local SCU Support Collection Unit..

The SCU will reach out to the party in question in order to obtain the necessary signature.
