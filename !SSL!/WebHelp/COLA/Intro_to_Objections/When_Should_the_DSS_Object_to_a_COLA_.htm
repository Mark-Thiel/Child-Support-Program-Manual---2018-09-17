When Should the CSEU File an Objection to a COLA?

When districts receive the Adjusted Order of Support in TA Temporary Assistance. "Temporary Assistance" includes Family Assistance (FA) and Safety Net Assistance (SNA). and Title IV-E foster care cases for which support rights have been assigned, districts should review ASSETS Automated State Support Enforcement and Tracking System (ASSETS). and other documentation to determine whether or not the COLA Cost-of-Living Adjustment, sometimes called a Cost-of-Living Allowance. order would bring the current support amount to a CSSA Child Support Standards Act, enacted 15 September 1989. level.

If it does not, the district should complete and file an objection and affidavit of service. This document must be generated using ASSETS Document Generation.

A sample of the Objection petition can be viewed at: http://www.courts.state.ny.us/forms/familycourt/pdfs/4-19.pdf
