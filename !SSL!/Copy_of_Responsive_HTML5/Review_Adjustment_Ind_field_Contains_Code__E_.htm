Code "E" (Error) Appears in Review Adjustment Ind field

 

If the Review Adjustment Indicator field contains Code "E" (Error):

 

A Code “E” (Error) in the Review Adjustment Ind field will stall the COLA for this case.

It prevents the COLA process from moving forward.

 

 

The “E” code indicates that two or more COLA-eligible ledgers on this account have conflicting dates in the Court Order Date fields (on the Obligation Tab screen).

 

The court order dates on all COLA-eligible ledgers must be identical, and they must match the date shown in the Review & Adjustment Date field.
