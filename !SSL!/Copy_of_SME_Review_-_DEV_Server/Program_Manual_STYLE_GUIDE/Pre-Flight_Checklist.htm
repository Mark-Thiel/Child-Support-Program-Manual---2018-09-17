Pre-Flight Checklist

(You can check or uncheck the boxes, and/or print out the checklist. (Boxes checked onscreen will appear checked when printed out, as well.)

Accessibility and ADA Compliance Pre-Flight

    1.  All hyperlinks have alt text and screen tip.

    2.  All images have alt text and screen tip.

    3.  Browse and tab sequences are correct.

    4.  Print output is generated to .docx file, then cleaned up for page breaks, etc., and output to PDF.

    5.  PDF is run through the Adobe tool to make it Section 508 compliant.

Graphics Pre-Flight

    1. Resolution:  Graphics are at resolution of 300 dpi.

    2. Size:  Graphics are set to width of not more than 750px

    3.  Graphics have border: color black, thickness 1px

    4.  Alt-text:  All graphics have alt-text and screen tips (this is for ADA compliance)

    5.  Centered:  All graphics are centered on the page

    6.  Meaningful names:  All graphics have meaningful names in the Project Manager pod -- not just names like image078, image079, etc.

Topic Pre-Flight

    1.  Links -- All links are live.

    2. Links -- All external links open in a new window (i.e. a new tab in the browser).

    3.  Index -- Any words in topic that should be indexed?

    4. Screen Names, Field Names, Legislation titles -- formatted with correct character styles

Global Pre-Flight

    1.  Glossary -- run global sync of Glossary Hotspot Wizard

    2.  Resource Manager Sync -- run global sync of Resource Manager files

    3.  Image Mover -- run script to move images from project root folder to topic folder (purchase from Weelden for 9.99 Euros)

    4.  Roadmap Process Diagrams -- All links are built into map?

    5.  Image Clean-Up script (free from Weelden's site.) - run the script

    6.  When you delete an image from a topic, RoboHelp does not delete the image from disk. This free script will find unused images and (optionally) remove them.

    7.  Troubleshooting topics: Make sure all topics are indexed with "troubleshooting" index tags

    8.  Roadmaps:  All roadmap pages are indexed with "Roadmap" (case sensitive) index tag

    9.  Browse Sequences:  Do we have browse sequences for all sections?

    10.  PDF Output:  Chapter must be output to MS Word doc first, then cleaned up (page breaks, etc.), and then output from MS Word to Adobe PDF.
