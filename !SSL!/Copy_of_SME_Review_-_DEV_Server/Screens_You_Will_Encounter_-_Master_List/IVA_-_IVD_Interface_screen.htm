Worker Alerts > IVA Mode > IV-A - IV-D Interface tab

In ASSETS, go to: Home > Tasks > Worker Alerts > IVA Mode > IVA - IVD Interface screen

Clicking a message in the "Comment" column will summon a pop-up dialog box detailing actions required on the part of the worker.

The IVA - IVD Interface Report is a daily report that serves as a bridge between WMS and ASSETS.

A case appears on the IVA - IVD Interface Report if the IV-D Indicator field in the WMS case record has been set with a code of Code Y, Code X, or Code D.

Code Y = Yes, the case is eligible for child support services.

Code X = The case is eligible for child support services, but this code also indicates to TA Unit workers that the child support pass-through payment will not automatically be issued for the case in question.

Code D = This is a Medicaid case that requires only paternity establishment.

It is important that the TA worker enters the correct IV-D Indicator code. Otherwise, some cases may not appear on the IVA - IVD Interface Report as they should, while other cases may appear when they actually should not.
