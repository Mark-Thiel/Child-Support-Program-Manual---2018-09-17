Manually Enter a New Respondent Address Into Case Record

PF/NCP addresses found through non-automated means or through the Social Security Administration's database must be entered into the case record manually by the worker.

The worker can manually add a new PF/NCP address into the case record by doing the following:

    1. Go to Summary Mode > Resp Addr tab.

    2. Click on the Add New Record button at the bottom of the screen.

The Summary Mode > Resp Addr Details tab appears.

    1. Enter the PF/NCP's address.

    2. For Do You Want Postal Clearance Submitted? select the Yes radio button.

    3. Click the Submit button.

The new PF/NCP address has now been entered into the case record, and a PCL form will be sent from the Processing Center to the postmaster for that address.

The postmaster will return the completed PCL to the Processing Center.  

The Processing Center will then update the Address Verification Indicator field with the appropriate Address Verification Code.
