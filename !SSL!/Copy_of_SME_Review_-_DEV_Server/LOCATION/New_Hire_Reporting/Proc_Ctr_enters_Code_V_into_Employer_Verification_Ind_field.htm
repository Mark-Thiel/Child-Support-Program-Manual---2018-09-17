Proc Ctr enters appropriate code into Employer Verification Ind field

When an employer is verified via a WHBR:

    1. The Processing Center sends a WHBR to the employer needing verification.

    2. The employer returns the completed WHBR to the Processing Center.

    3. The Processing Center updates the Employer Verification Ind field on the Case Mode > Resp Emp tab with Code V or Code N.

    4. The Processing Center updates ASSETS during the next routine batch processing session.

    5. ASSETS closes the F Group with the appropriate code.
