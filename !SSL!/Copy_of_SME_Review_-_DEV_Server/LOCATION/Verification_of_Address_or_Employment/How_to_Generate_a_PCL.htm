How to Generate a Postal Clearance Letter/Address Information Request form

 

All PF/NCP addresses must be verified via a PCL form.

  1. Summary

      1. The worker needs to verify an address.

      2. The worker generates an PCL form via ASSETS.

      3. ASSETS sends data files to the Processing Center during nightly batch processing.

      4. The Processing Center prints the PCL form and mails it to the Postmaster who serves the address needing verification.

      5. The Postmaster fills out the PCL form and mails it back to the Processing Center.

      6. Upon arrival at the Processing Center, the PCL form is automatically scanned and imaged, and the data in the form is captured.

      7. The captured data is fed into the computer system run by the Processing Center.

      1. The Processing Center sends data files to ASSETS during the next batch processing session.

      1. ASSETS receives data files from the Processing Center, and updates its own (corresponding) datafields accordingly.

    *  If the worker checks the box for Do you want a copy of the completed document returned to your district? an image of the actual PCL form completed by the Postmaster is available to the CSEU worker via ECS, accessible from the ASSETS homepage.

     

  2. What is a PCL form?

    The PCL form is used to ask a Postmaster to confirm or deny that a certain person (i.e. a PF/NCP in a case) receives mail at a certain address, and/or to provide additional information, such as a forwarding address, if applicable.

  3. When to use a PCL form

    When you have an address for a PF/NCP, but are unsure whether it is current or valid, you need to verify the address via an PCL form.

    You generate the PCL form through ASSETS.

  4. How to generate a PCL in ASSETS

    1. In ASSETS, open a case, then go to:  Summary Mode > Resp Addr  tab.

    2. On the Respondent Addresses table, click on UPD in the row containing the address you want to verify.

    The Respondent Address Details screen appears. Click the Yes radio button if you want a copy of the PCL returned to your district.

    1. Click Submit.

    A PCL will be mailed to the Postmaster.

    * Note: A PCL can also be initiated by opening a V Group on the Case Mode > Status Information tab.

  5. When the Postmaster receives a PCL form

    When the Postmaster receives the form, he or she will indicate one of the following on the PCL form:

      Mail is delivered to the address given

      Mail is forwarded to a new address

      [PF/NCP is] Not Known at address given

      [PF/NCP] Moved, left no forwarding address

      No such address

      Other (Specify)

    The Postmaster will also provide a street address, if available, for a PF/NCP who has a PO Box, and a new address, if the PF/NCP has moved and provided one.

  6. To view the status of PCL queries

    You can view the Address Verification Indicator (Ver Ind) code on the Summary Mode > Resp Addr tab. This will tell you the status of the PCL.
