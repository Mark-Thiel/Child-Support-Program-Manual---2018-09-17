What is a Support Petition?

A Support Petition is a written application that is filed in family court requesting that the court issue an order for child support. A Support Petition can be generated in your local district by using the Document Generation module of ASSETS. Information in the case record (in ASSETS) will auto-populate the petition.

 

Preparing a Support Petition

To prepare a Support Petition, do the following:

  1. In ASSETS, go to: Home > Modules > Document Generation > NYS Family Court tab

The NYS Family Court tab appears.

  1. Select the Petitioner (i.e. who is bringing the action?)

  2. Select Petition Type.

  3. Select Paper Petition Filing or Electronic Petition Filing.

  4. Click the Generate button

The Support Petition Assessment screen appears.

The top of this screen will be prepopulated with case information such as Case ID and the names, SSNs, DOB, and ID numbers of the parties.

Documents generated are saved to the document log

Documents generated through the Document Generation module in ASSETS are saved in PDF format to the case-specific document log, with the date and document type recorded and displayed, and can be easily retrieved for the life of the case.

 

Saving a document to the log automatically opens a case status group displaying the type of document and the date the document was generated.
