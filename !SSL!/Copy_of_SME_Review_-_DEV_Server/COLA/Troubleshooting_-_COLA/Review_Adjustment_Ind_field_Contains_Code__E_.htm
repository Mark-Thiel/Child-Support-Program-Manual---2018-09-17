Code "E" (Error) Appears in Review Adjustment Ind field

A Code E (Error) in the Review Adjustment Ind field will prevent the COLA process from moving forward.

The Code E indicates that two or more COLA-eligible ledgers on this account have conflicting dates in the Court Order Date fields (on the Summary Mode > Obligation Tab screen).

The court order dates on all COLA-eligible ledgers must be identical, and they must match the date shown in the Review & Adjustment Date field.
