If a modified order is issued

The court, upon hearing the objections, may issue an order which increases or decreases the amount owed or issue an order that remains the same, based on the CSSA.

If a new modified Support Order is issued by the court:

    • The worker clears the Adj. Order Objection Date field and enters the Court Order Date for all affected ledgers

    • The Court Order Date is reflected on the new Court Order itself, which is sent by the court to the CSEU
