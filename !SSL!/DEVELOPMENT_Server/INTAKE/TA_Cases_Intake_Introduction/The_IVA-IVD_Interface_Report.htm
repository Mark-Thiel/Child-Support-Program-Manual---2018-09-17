The IVA-IVD Interface Report

Referrals from the TA Temporary Assistance. "Temporary Assistance" includes Family Assistance (FA) and Safety Net Assistance (SNA). Unit appear on the IVA-IVD Interface Report. The Report is refreshed every day and should be reviewed every day.

It can be found on the ASSETS Automated State Support Enforcement and Tracking System (ASSETS). system by going to: Home > Tasks > Worker Alerts > IVA Module > IVA - IVD tab

Best Practices for Using the IVA-IVD Interface Report:

    1. Check the IVA-IVD Interface Report every day for new TA, MA, or FC Foster Care cases.

    2. Build new cases promptly.

    3. If you cannot locate the hard copy referral A request sent to a CSEU from another jurisdiction or a non-IV-D agent or agency asking that a child support order be established. (LDSS-4882 "Information about Child Support Services and "Application/Referral for Child Support Services" intake form. (Form LDSS-4882)) form, check to see whether the TA or MA Unit has cancelled the case/referral.

    4. For cases that do not require child support services, reject the case.

    5. For cases that do NOT require child support services, contact the TA or MA unit to have them change the IV-D Indicator so that you don't continue getting messages regarding this case.

What Triggers a Case Appearing on the IVA-IVD Interface Report?

A case will appear on the IVA-IVD Interface Report if the IV-D Indicator field of the WMS Welfare Management System. The WMS system used by the TA units supplies referrals and other information to the ASSETS system used by Child Support units.. The relationship is one-way only: WMS can update some information in the ASSETS system (CP residential addresses, for example), but the ASSETS system cannot update any datafields in the WMS system. case record has been set with a code of Code D, Code X, or Code Y.

Code D = Medicaid case requiring only paternity establishment.

Code X = Eligible for child support services, and pass-through payment In TA or MA cases, the first UP TO $100 (for one child) or UP TO $200 (for two children) colllected each month is sent directly to the CP. This is called a "pass-through" payment. will not be automatically issued for this case.

Code Y = Case is eligible for child support services.

It is important that the TA worker enters the correct IV-D Indicator into the WMS case record, since that's how the Child Support Program becomes aware of those cases.

A case with a Code D, Code X, or Code Y in the IV-D Ind field in the WMS case record will appear on the IVA-IVD Interface Report in ASSETS (see screenshot below).
