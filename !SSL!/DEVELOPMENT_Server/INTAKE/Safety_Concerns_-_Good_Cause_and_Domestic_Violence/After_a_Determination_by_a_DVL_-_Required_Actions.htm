After a Determination by a DVL:  Required Actions

Source:  12-ADM-03

If a Full Waiver is granted, the worker must:

    1. Set the FVI Family Violence Indicator. The FVI is a designation in the Federal Case Registry. It is placed on a participant in a case or order, and indicates that the participant is at risk of child abuse or domestic violence. The FVI is used to prevent disclosure of the location of a party or a child believed by the state or tribe to be at risk of family violence. The Federal Case Registry (FCR) uses this term to identify a person involved in a family violence case or order in another state. indicator code to value Code 1.

    2. Cease all case and enforcement activity.

If a Partial Waiver is granted, the worker must:

    1. Set the FVI indicator code to Code 2.

    2. Proceed with case-building, but with certain precautions

Setting these indicator codes also serves to provide notification of family violence on the FCR Federal Case Registry [of Child Support]. A national database of information on all people with IV-D cases and people with Non-IV-D orders that were entered or changed on or after October 1, 1998. The FCR receives this case information on a daily basis from the State Case REgistry located in each state, and proactively matches it with previous submissions to the FCR and with employment information contained in the National Directory of New Hires (NDNH). Any successful matches are returned to the appropriate state(s) for processing. The FCR and the NDNH are both part of the Federal Parent Locator Service maintained by OCSE..

The CSEU Child Support Enforcement Unit, to the extent required by such waiver, must forego any enforcement actions or activities on the case with respect to the children of the NCP Non-Custodial Parent. Sometimes called the "absent parent."/PF Putative Father. This is a person who is presumed to be the father of a child, though there may not yet be hard evidence to support that presumption. while the waiver is in effect.

Family Violence Indicator (FVI) Code

Meaning

Blank

No Family Violence Indicator set on case at FCR.

System set when prior value is Code X. Manually set when prior value is Code Y.

V

Family Violence Indicator sent to the FCR.

System set when prior value is Code Y.  Manually set when prior value is Code X.

X

Request removal of FVI code  that has been set on the case at the FCR.  

Manually set only when prior value is Code V.

Y

Request notification of family violence to the FCR.

Manually set only when prior value is blank.

For more information, refer to the Dear Colleague letter dated August 10, 2007 regarding Family Violence Waiver indicator values.
