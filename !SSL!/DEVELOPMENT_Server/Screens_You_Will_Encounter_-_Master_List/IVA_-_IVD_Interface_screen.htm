IV-A - IV-D Interface screen

In ASSETS Automated State Support Enforcement and Tracking System (ASSETS)., go to: Home > Tasks > Worker Alerts > IVA Mode > IVA - IVD Interface screen

The IVA - IVD Interface Report is a daily report that serves as a bridge between WMS Welfare Management System. The WMS system used by the TA units supplies referrals and other information to the ASSETS system used by Child Support units.. The relationship is one-way only: WMS can update some information in the ASSETS system (CP residential addresses, for example), but the ASSETS system cannot update any datafields in the WMS system. and ASSETS.

A case appears on the IVA - IVD Interface Report if the IV-D Indicator field in the WMS case record has been set with a code of Code Y, Code X, or Code D.

Code "Y" = Yes, the case is eligible for child support services.

Code "X" = The case is eligible for child support services, but this code also indicates to TA Temporary Assistance. "Temporary Assistance" includes Family Assistance (FA) and Safety Net Assistance (SNA). Unit workers that the child support pass-through payment In TA or MA cases, the first UP TO $100 (for one child) or UP TO $200 (for two children) colllected each month is sent directly to the CP. This is called a "pass-through" payment. will not automatically be issued for the case in question.

Code "D" = This is a Medicaid case that requires only paternity establishment.

It is important that the TA worker enters the correct IV-D Indicator code. Otherwise, some cases may not appear on the IVA - IVD Interface Report as they should, while other cases may appear when they actually should not.
