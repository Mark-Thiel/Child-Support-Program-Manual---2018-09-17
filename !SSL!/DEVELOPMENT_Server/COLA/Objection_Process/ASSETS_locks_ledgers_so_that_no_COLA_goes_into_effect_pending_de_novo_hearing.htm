ASSETS locks ledgers so that no COLA goes into effect pending de novo hearing

ASSETS Automated State Support Enforcement and Tracking System (ASSETS). locks the ledgers so that no COLA Cost-of-Living Adjustment, sometimes called a Cost-of-Living Allowance. can go into effect pending a decision at a hearing.

The ledgers will remain locked until the worker clears the Adj. Order Objection Date field is cleared.
