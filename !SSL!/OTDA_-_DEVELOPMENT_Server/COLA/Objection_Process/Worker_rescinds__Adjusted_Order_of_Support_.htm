Worker rescinds "Adjusted Order of Support"

The worker In the Child Support Services Operations Manual, the term "worker" (in upper- or lower-case) refers to personnel working in any child support function. rescinds the "Adjusted Order of Support" by entering Code "R" (Rescind Revoke, cancel, or repeal (a law, order, or agreement).) in the COLA Cost-of-Living Adjustment, sometimes called a Cost-of-Living Allowance. Request Indicator field on the Cost of Living Adjustment (COLA) Information Edit screen.

    1. Go to the Obligation Tab screen (See screenshot below)

    2. Click on the "Update COLA" button in the bottom right of the screen

    3. The Cost of Living Adjustment (COLA) Information Edit screen appears (See screenshot below)

    4. Enter Code "R" (Rescind) in the COLA Request Indicator field

In ASSETS Automated State Support Enforcement and Tracking System (ASSETS)., go to:  Home > Module > Inquiry > [enter search criteria] > Name Search Results screen > [double click on: name of client/case > Parties to Action tab > [click on:] Obligation Tab > "Update COLA" button > Cost of Living Adjustment (COLA) Information Edit screen

 

Clicking on the "Update COLA" button in the screen above brings you to the screen below. Use the dropdown menu under "Updatable COLA Data Elements" to select "COLA Request Indicator," then select the desired code to enter into the COLA Request Indicator field.
