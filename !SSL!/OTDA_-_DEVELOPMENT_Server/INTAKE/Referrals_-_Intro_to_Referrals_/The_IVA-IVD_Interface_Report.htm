What is the IVA-IVD Interface Report, as opposed to the IVA-IVD interface?

The IVA-IVD Interface Report should not be confused with the IVA-IVD interface itself.

The IVA-IVD Interface Report refers to the screen in ASSETS that displays a subset of all the data sent from WMS to ASSETS in daily batch processing between the two systems.  In other words, the interface report displays only some of the (columns of) data contained in the data interface/exchange itself.

The IVA-IVD Interface

Each day, WMS sends files to ASSETS. These files do not contain every mailing address for every CP Custodial Parent. This is the parent with whom the child is living. However, the "Custodial Parent" could be a guardian such as a grandparent, foster home, or other person or entity with whom the child is living, and who is taking care of the child.. Instead, it contains CP mailing addresses that have been updated or modified in the WMS system.

The IVA-IVD Interface serves as a bridge of information (an interface) between IV-A Title IV-A of the Social Security Act pertains to Temporary Assistance for Needy Families (TANF); Temporary Assistance (TA), and Safety Net Assistance (SNA). (Temporary Assistance) Units and IV-D (Child Support) Units -- or, more specifically, between the Welfare Management System (WMS) used by TA Temporary Assistance. "Temporary Assistance" includes Family Assistance (FA) and Safety Net Assistance (SNA). Units and the ASSETS system used by the child support workers.

Some information entered into WMS will automatically populate or refresh some fields in ASSETS. But the relationship between WMS and ASSETS is a one-way street: changes made in ASSETS will not update or modify any fields in the WMS system, but some changes made in the WMS system (CP mailing address, for example) will update or modify data fields in the ASSETS system automatically, under certain conditions.

Example:

When ASSETS receives from WMS a file containing a mailing address for a CP, ASSETS reads the file and compares it to the address that ASSETS already has.  If the CP mailing address in ASSETS is more than 60 days old, and the CP mailing address from ASSETS is different from the one that ASSETS already has, ASSETS will update the CP mailling address to match the mailing address contained in the file sent by WMS.

What Triggers a Case Appearing on the IVA-IVD Interface Report?

A case will appear on the IVA-IVD Interface Report if the IV-D Indicator field of the WMS case record has been set with a code of Y, X, or D.
