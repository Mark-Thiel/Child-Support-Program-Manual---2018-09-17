Does worker have enough info to build the case and account?

Yes -- build the case.

No -- Retrieve the ACS Account Creation Summary. The ACS is an ASSETS-generated court report. It can provide info including the parties' and children's names, addresses, SSNs, DOBs, employment statuses, and name of NCP's employer. The ACS can be retrieved via the UCMS (Universal Case Management System). Report.
