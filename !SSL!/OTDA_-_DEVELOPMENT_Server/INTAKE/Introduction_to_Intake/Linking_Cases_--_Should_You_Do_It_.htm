Linking Cases -- Should You Do It?

When should you link cases?

Please exercise caution when considering whether to link cases.  Linking of cases should not be a default stance. Once linked, cases cannot be unlinked.

Cases should be linked when -- and ONLY when -- all parties in both cases are the same.

Example:

A single case can contain a maximum of 18 ledgers. Potential adverse effects of linking cases

When cases are linked, any case remarks entered in one case record will appear in the case records of all cases linked to it.

This can make it difficult for a CSEU worker viewing these remarks to know to which case (or party?) the remarks are referring to.

Example:

If there are cases regarding multiple NCPs linked to a single CP, for example, remarks referring to "the NCP" will be confusing, since the worker would not know which NCP the remarks are referring to.
