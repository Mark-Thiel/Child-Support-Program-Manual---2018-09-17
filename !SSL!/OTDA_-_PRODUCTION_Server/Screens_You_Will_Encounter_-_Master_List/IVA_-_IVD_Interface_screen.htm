IV-A - IV-D Interface screen

In ASSETS Automated State Support Enforcement and Tracking System (ASSETS)., go to: Home > Tasks > Worker Alerts > IVA Mode > IVA - IVD Interface screen

The IVA - IVD Interface Report is a daily report that acts as a bridge between the WMS Welfare Management System. ?MT - Medicaid and TA cases are in the WMS System? Need explanation. and ASSETS.

A case appears on the IVA - IVD Interface Report if the IV-D Indicator field in the WMS case record has been set with a code of "Y", "X", or "D."

Code "Y" = Yes, the case is eligible for child support services.

Code "X" = The case is eligible for child support services, but this code also indicates to TA Temporary Assistance. "Temporary Assistance" includes Family Assistance (FA) and Safety Net Assistance (SNA). Unit workers that the child support pass-through payment will not automatically be issued for the case in question.

Code "D" = This is a Medicaid case that requires only paternity establishment.

It is important that the TA worker In the Child Support Services Operations Manual, the term "worker" (in upper- or lower-case) refers to personnel working in any child support function. enters the correct IV-D Indicator code. Otherwise, some cases may not appear on the IVA - IVD Interface Report as they should, while other cases may appear when they actually should not.
