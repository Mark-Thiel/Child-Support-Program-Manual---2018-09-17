Postal Clearance Letter (PCL)

A Postal Clearance Letter (PCL Postal Clearance Letter) is used to confirm an NCP's or PF's address and establish it in ASSETS Automated State Support Enforcement and Tracking System (ASSETS). as a verified address.

When you have an address but are unsure if it is current or valid, you need to use ASSETS to generate a PCL. The PCL is mailed out from the Processing Center to the Postmaster for the post office that handles the mail for that address.

The postmaster will return the PCL to the Processing Center with either an indication that the address is valid or an indication that the address is not valid.

An image of the returned document is captured and saved by the Processing Center.

The Processing Center will notify the local CSEU Child Support Enforcement Unit of the results of the via ECS Electronic Communications System, used for communications between the Processing Center and local SCUs., and the notification will include the image of the returned PCL.
