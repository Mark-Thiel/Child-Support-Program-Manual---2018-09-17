Billing Switch Codes

The Billing Switch code is system-set during the initial account-building process, and is based upon the FIPS Code.  However, after the account is built, the user can change the Billing Switch via the Account Update process.

Code

Description

1

Bill the PF/NCP, include delinquency warning, if applicable, and notify non-Office of Temporary Disability and Assistance (OTDA) client of delinquency

2

Do no bill PF/NCP or notify non-OTDA customer

3

Bill PF/NCP, do not include delinquency warning or notify non-OTDA customer

4

Bill PF/NCP, include delinquency warning, if applicable,; and notify non-OTDA customer of delinquency income withholding amount is greater than zero

5

Do not bill the PF/NCP or notify non-OTDA customer income withholding amount is greater than zero

6

Bill the PF/NCP, do not include delinquency warning or notify non-OTDA customer income withholding amount is greater than zero

7

Bill regardless of delinquency switch

The following are the rules regarding the system of the Billing Switch code:

    1. If FIPS position 1,2,3 = 136 and position 4-6 NOT = user county code then set bill switch value = 5

    2. If the FIPS position 1,2,3 = 136 and position 4-6 = user county code then set bill switch value = 4

    3. If FIPS position 1 = 1 and 2-6 = other valid value then set bill switch = 5.

    4. For any other FIPS code, the system will default bill switch to 4.
