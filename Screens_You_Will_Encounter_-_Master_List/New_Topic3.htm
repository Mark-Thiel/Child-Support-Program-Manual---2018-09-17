ASSETS Basic Search

Go to Home > Modules

 

 

You can search by any of the following:

 

  • Name of Child

  • Name of Customer

  • Name of PF/NCP

The system will return hits for the name entered, and also for any names that are similar. For example, if you search for "Joe Smith," ASSETS will return hits for:

  • Joe Smith

  • Jo Smith

  • Joseph Smith

  • Josef Smith

  • Jose Smith

  • Jo Smith

  • Name of Child

  • Name of Customer

  • Name of PF/NCP
