    • Documents generated through the Document Generation module in ASSETS are saved in PDF format to the case-specific Document Log, with the date and document type recorded and displayed, and can be easily retrieved for the life of the case.

    • Excel spreadsheets cannot be saved to the Document Log.  However, it is possible to embed an Excel spreadsheet in an MS Word document, and save the MS Word document to the document log as a PDF.

    • Each document generated sets a status on the Case Mode > Status tab to record the date and type of document generated in the case record.
