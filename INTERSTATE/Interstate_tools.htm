Tools

To carry out your responsibilities for interstate cases, you'll need to use:

  • Interstate Case Assistant, accessible from the Interstate Cases page of ERS. For any given case, this is what you use to identify how to proceed, what documents to use, where to send those documents, special considerations to be aware of, and the status code you need to set in the case record.

  • ASSETS Document Generation > UIFSA tab. This is what you use to access, prepare, and print the forms that are a required element of case activity. You must have a specific case record open in ASSETS to be able to access the Document Generation module.

  • CSENETINFOREPORT, accessible through COLD. This is what you use to view messages from other states received via CSENet. CSENet is an electronic interface managed by Federal OCSE that enables the exchange of messages between the systems of participating states. New York is a participating state, but district users in New York do not directly access CSENet. Rather, messages from New York to other states are automatically generated when appropriate based on case data in ASSETS, and messages to New York from other states are displayed on this report.

  • CSENETINFOREPORT Action Guide, accessible from the Interstate Cases page of ERS. This is what you use to identify what you need to do in response to messages on the CSENETINFOREPORT.

You'll also need to use the same tools you use as a general part of case processing an caseload management for local cases (ASSETS, hardcopy files, ECS, other COLD reports, etc.).
