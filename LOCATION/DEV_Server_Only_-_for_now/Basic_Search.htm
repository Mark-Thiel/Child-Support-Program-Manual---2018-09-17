Basic Search

The Basic Search is the default screen in Inquiry.

With a basic search, you can search by:

    • Name;

    • Respondent or Client ID; or

    • Social Security number

 When you search by name, the system will search for not only the name you provided in the search field, but also for any names similar to the name entered or which are variations of the name entered (i.e. Joseph, Joe, Jose).
