Long-Arm Jurisdiction

If we have long-arm jurisdiction, the worker should file a regular paternity and/or support petition, and request certified mail.

If we have reciprocity with the country in which the Respondent is presumed to reside AND have a possible address for the Respondent, the worker must do the following:

(Note We must already have an address for the Respondent in the country in question, since we cannot open a location search in another country.)

    1. Build the case with worker code OOS.

    2. Enter address on RA screen.

    3. SI screen:  set V000 for the address and set R001.

    4. RR: give details of situation regarding where the Respondent lives, that we have reciprocity, etc., and that UIFSA needed if postal is positive.

* If postal is positive, we will file UIFSA. If the postal is negative, proceed as below (i.e. no long-arm, no reciprocity, or postal to foreign country is negative):
