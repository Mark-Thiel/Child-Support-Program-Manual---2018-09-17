Essential Screens in the Location Process

 

Case Mode > Status tab

    • Open an L Group

    • Add a new status to an L Group

    • Update a status

    • Delete a status

    • See a neatly summarized longitudinal history of location requests on the case

Case Mode > Resp Emp tab

    • Add new employer record

    • Update an employer record

    • Delete an employer record

    • Add health insurance information

Summary Mode > Resp Addr tab

    • Add new PF/NCP address

Worker Alerts > Enforcement Mode > PLS tab

    • See notifications of information found through automated location searches

    • Click on notifications to get required actions
